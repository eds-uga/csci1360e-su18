CSCI 1360E

Foundations for Informatics and Analytics

Welcome to the Summer 2018 rendition of CSCI 1360E.

Announcements

All announcements will be made in the Slack chat. This is also the appropriate venue for asking and answering questions.

Items of Note

This course will be conducted entirely online. If you're on campus over the summer I'm happy to schedule in-person meetings, but this is completely optional and up to you.

You are not expected to know any programming before taking this course; in fact, it's aimed specifically at people with no prior programming experience.

We will be using Python 3 for the course assignments. You are not required to install Python on your local computer; however, if you want to, there will be instructions for doing so later in the course.

Course materials will be linked from the lecture schedule and permanently archived in the GitHub repository.
